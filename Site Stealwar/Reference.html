Mécaniques de jeu

Dans StealWar, vous devrez maîtriser plusieurs mécaniques de jeu pour progresser. Voici un aperçu des principales mécaniques :

  • Ramassage d'objets : Appuyez sur Clic droit pour ramasser des objets lorsque vous les voyez.
  • Utilisation des compétences : Chaqu'un des personnages a des charactéristiques uniques. Utilisez-les stratégiquement pour avancer dans le jeu.
  • Interaction avec l'environnement : Appuyez sur les fléches pour interagir avec dans l'environnement.

Conseils et astuces

Voici quelques conseils pour vous aider à mieux jouer :

  • Explorez chaque zone minutieusement pour trouver des objets cachés.
  • Utilisez les compétences de vos personnages de manière complémentaire pour surmonter les défis.
  • Gardez un œil sur votre inventaire et utilisez vos ressources judicieusement.

FAQ

Vous avez des questions ? Voici les réponses aux questions les plus fréquemment posées :

  • Quels sont les contrôles du jeu ? Vous pouvez trouver la liste complète des contrôles dans le menu des paramètres.
  • Où puis-je obtenir de l'aide supplémentaire ? Rejoignez notre serveur Discord pour obtenir de l'aide et discuter avec d'autres joueurs.
